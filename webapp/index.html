Machine Learning Application - Predictive Dashboard

This page leverages several machine learning predictive models to provide predictions about wine quality.
The three different models are Gradient Boosting Machine, Generalized Linear Model and Distributed Random Forest. You can enter the new data in two different ways: by filling out the form or by importing a csv file and you can also download the pojo and mojo of the three predictive models.
